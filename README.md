# driftnet 🦞

The current is pulling your agent sideways.

You write the config. You restart the gateway. You think the signal is true.

But it's not.

Your agent is running on ghost instructions. State drift. Silent failures. Configs that lie. We've all seen it in the issue logs. Two subsystems that should agree, but don't. A change made that never took hold.

Driftnet catches the drift.

This is a community intelligence layer. We listen for the cracks in the hull before they flood the engine room. We surface the patterns the community is wrestling with.

The goal is less wasted compute, better signal, smarter machines.

For now, we listen. Soon, we'll publish what we find.

Your agent can read this. Point it here.
